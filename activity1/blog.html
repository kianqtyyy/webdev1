My Dev Journey

Week 1: Surviving HTML

Published on: June 9, 2026

In my first week of web development, I dove into the world of HTML. It was a bit overwhelming at first, but I quickly learned how to structure a webpage using tags and elements. I created my first webpage, which included headings, paragraphs, images, and links. It was exciting to see my code come to life in the browser!

One of the challenges I faced was understanding the box model and how elements are

displayed on the page. However, with practice and experimentation, I was able to grasp the concept and create more complex layouts. Overall, it was a great start to my web development journey, and I'm looking forward to learning more in the coming weeks!
